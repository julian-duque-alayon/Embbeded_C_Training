# ESP32 Project Template

This is a base template for **ESP32** development using **ESP-IDF**.

## How to use:
1. Copy this folder to a new location.
2. Open the folder in VS Code.
3. Use the **ESP-IDF** extension to Configure, Build, and Flash.

## Build manually:
```bash
idf.py build
idf.py -p [PORT] flash
idf.py -p [PORT] monitor
```
